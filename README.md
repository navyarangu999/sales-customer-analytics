# sales-customer-analytics
Sales &amp; Customer Analytics using Python and SQL End-to-end data analytics project where raw sales data is cleaned using Pandas, stored in a SQL database, and analyzed with business queries to generate actionable insights.


This project demonstrates a real-world analytics workflow using Python and SQL.
It covers data ingestion, cleaning, database design, and analytical queries.
The goal is to answer business questions like revenue trends and top customers.
Built with Python (Pandas) and SQL.
